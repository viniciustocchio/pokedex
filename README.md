# Pokédex App
***
## Introduction
---
The Pokédex is a small web application with HTML, CSS, and JavaScript that loads data from an external API to load information about 150 pokemons.

---
## Technologies used
---
* `HTML`
* `CSS`
* `JavaScript`
* `Bootstrap`
* `Fetch polyfill`
* `Promise polyfill`
